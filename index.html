Performer is integrated set of tools made for load, stress and preformance tests in telecommunication networks

Blender

The Blender is external dynamic data generator. The tool is combing structured data and randomly generate inputs then the simulation can be perform to desired behaviour.

Templer

The templer is template file manipulator able to make distributed simplators configuration. The templer is made to work with Jmeter .jmx files , Seagull .xml files and SoupaUI .xml files.

Modeler

The Modeler is tool symulate entire traffic network, running simulation from multiple nodes and moniting each od running processes

Tracer

The Tracer is tool desgined to catche .pcap files from multiple nodes. Enchanded version of this tool is able to do DIAMETER, CAP,MAP analyse.

Logger

The Logger is simple tool desgined to collect and parse system and application log files. Also, the tool are able to collect alarms from nodes.

Repoter

The reporter tool maks test reports, presenting graps and calulating statis and colleting logs

Utilaser

The Utilaser is simple multi node system monitoring tool. It is designed to collect, present and browser system activity reports from multiple nodes
